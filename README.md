# JokerDBDTracker

Локальный трекер стримов JokerDBD с встроенным плеером, эффектами, прогрессом, заданиями и ачивками.

Текущая версия: `v1.2.3`

## Что нового в `v1.2.3` (по сравнению с `v1.2.2`)

### Плеер и эффекты

- Добавлено удобное управление плеером с клавиатуры при открытой панели эффектов:
  `Space` (пауза/пуск), стрелки (перемотка/громкость) работают даже при работе с эффектами.
- Убран конфликт `Space` с чекбоксами эффектов: пробел больше не случайно переключает галочку эффекта.
- Для эффекта `Fisheye` добавлен мини-пад для настройки центра линзы мышкой (перетаскивание точки).
- Добавлено отображение координат центра `Fisheye`, исправлена инверсия по вертикали.
- Сброс эффектов теперь корректно возвращает центр `Fisheye` в центр экрана.

### Пресеты в плеере (новый сценарий работы)

- Добавлена отдельная страница `Пресеты` в панели плеера (`Эффекты / Пресеты`).
- Добавлены готовые пресеты (в том числе более “экспериментальные”).
- Добавлены кастомные пресеты: до `20` слотов с именами.
- Сохранение кастомного пресета теперь сохраняет текущее состояние эффектов и их настройки.
- После загрузки/применения пресета интерфейс возвращается обратно к панели эффектов.
- Исправлен ввод названия пресета: горячие клавиши эффектов больше не мешают печатать в поле.
- `Enter` в поле имени пресета сохраняет, `Esc` возвращает к эффектам.

### Интерфейс и удобство

- Обновлён визуальный стиль приложения (тёмная палитра, стеклянные элементы, более современный вид).
- Обновлены стили главной страницы, профиля, настроек и плеера в едином стиле.
- Переделан экран загрузки: более чистый вид, без лишних слоёв и артефактов по краям.
- Обновлены стили выпадающих списков (`ComboBox`): скругления, читаемость текста, корректная работа кликов.
- Исправлены проблемы отображения и взаимодействия с выпадающими списками в главном окне и плеере.
- Переработаны кастомные скроллбары (визуал и логика показа), исправлены артефакты/обрезания в списках.

### Профиль, ачивки и задания

- Добавлен счётчик часов, проведённых в программе (в профиле).
- Добавлено много новых достижений:
  за пресеты, кастомные пресеты, сессии с эффектами, серии дней, квесты, XP, престиж и т.д.
- Добавлены новые задания (в том числе задания, связанные с использованием пресетов).

### Исправления стабильности и багов

- Исправлены проблемы локализации и отображения текста в ряде экранов (главная/плеер).
- Исправлены случаи, когда панель эффектов/вкладка эффектов открывалась некорректно.
- Исправлен баг YouTube-плеера, когда при старте могла вылезать боковая панель и сдвигать видео.
- Исправлен залипший выбор элемента после клика по кнопке избранного.
- Исправлен `pitch`-эффект: изменение скорости в YouTube больше не ломает тон/звук.

## Основные возможности

- Каталог стримов и быстрый поиск
- Избранное и история просмотров
- Встроенный плеер с видео/аудио-эффектами
- Горячие клавиши для управления плеером и эффектами
- Пресеты эффектов (готовые и кастомные)
- Профиль, XP, уровни, престиж
- Ежедневные и недельные задания
- Локальные достижения и статистика просмотра
- Встроенное обновление приложения

## Установка

1. Открой релизы: `https://github.com/dragloner/JokerDBDTracker/releases`
2. Скачай свежий архив `win-x64`
3. Распакуй в удобную папку
4. Запусти `JokerDBDTracker.exe`

## Обновление

- Нажми кнопку обновления внутри приложения.
- Программа скачает обновление, покажет прогресс и перезапустится.
- В случае ошибки применяется безопасный откат, чтобы приложение не осталось в сломанном состоянии.

## Сборка (для разработчиков)

Требуется:

- Windows 10/11
- .NET SDK
- WebView2 Runtime

Команды:

```powershell
dotnet build JokerDBDTracker.sln -c Debug
dotnet build JokerDBDTracker.sln -c Release
dotnet publish JokerDBDTracker.csproj -c Release -r win-x64
```
