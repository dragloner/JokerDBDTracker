## Что это такое

**JokerDBD Tracker** — приложение для фанатов стримера JokerDBD. Встроенный плеер позволяет смотреть стримы и VOD-ы прямо в программе, накладывать на видео визуальные фильтры и аудио-эффекты в реальном времени, копить XP, выполнять задания и открывать ачивки. Плюс совместный просмотр с друзьями.

---

## Установка — пошагово для новичков

> Никакого программирования — всё просто.

**Шаг 1.** Перейди на страницу релизов:
👉 **[github.com/dragloner/JokerDBDTracker/releases](https://github.com/dragloner/JokerDBDTracker/releases)**

**Шаг 2.** В последнем релизе нажми на файл `JokerDBDTracker-vX.X.X-win-x64.zip` — начнётся скачивание.

**Шаг 3.** Открой скачанный архив. Нажми правой кнопкой → **«Извлечь всё...»** → выбери папку (например `C:\Programs\JokerDBDTracker`).

**Шаг 4.** Зайди в распакованную папку и запусти **`JokerDBDTracker.exe`**.

> ⚠️ Если Windows показывает предупреждение «Неизвестный издатель» — нажми **«Подробнее»** → **«Всё равно запустить»**. Это нормально для неподписанных приложений.

> ⚠️ Если появляется ошибка про **WebView2** — скачай его отдельно: [Microsoft WebView2 Runtime](https://developer.microsoft.com/ru-ru/microsoft-edge/webview2/)

**Готово!** Приложение запустится — войди под своим ником и смотри стримы.

## Что умеет приложение

### 🎬 Встроенный плеер
- Смотри стримы и VOD-ы JokerDBD прямо в приложении без браузера
- Полноэкранный режим, управление с клавиатуры
- Панель чата слева (YouTube chat replay / live chat)
- Панель комментариев снизу

### 🎨 Визуальные и аудио-эффекты в реальном времени
**15 визуальных фильтров:** ч/б, сепия, инверсия, контраст, темнота, насыщенность, сдвиг цвета, размытие, фишай, VHS, тряска, JPEG-артефакты, холодный тон, горизонтальное и вертикальное зеркало

**Аудио-эффекты:** усиление громкости, изменение тона и скорости, реверб, эхо, дисторшн, вобблинг, 3-полосный эквалайзер

### 🎛️ Пресеты эффектов
- 12 встроенных пресетов: JNightmare, JFire, JRelaxing, Retro VHS, Dream Glow, JVoid и другие
- До 20 кастомных слотов — сохраняй свои сборки

### 🔊 Звуковые эффекты
Воспроизведение звуков поверх стрима одной клавишей: Aura Farm, Смех, Psi Radiation, Don Mafia, Грусть, «Да уж точно»

### 🎯 Горячие клавиши
Все действия доступны с клавиатуры, все бинды настраиваются в настройках

### 👥 Watch Together
Совместный просмотр с друзьями по LAN или Radmin VPN. Хост создаёт комнату, гости подключаются по IP. Синхронизируются позиция, пауза/плей и все эффекты.

### 🏆 Профиль, XP и ачивки
XP за просмотр, уровни, престиж, ежедневные и еженедельные задания, достижения

### ⏱️ Таймкоды
Создавай метки в видео с тегами и заметками, переходи по ним одним кликом

## Горячие клавиши (по умолчанию)

| Клавиша | Действие |
|---------|----------|
| `H` | Скрыть / показать панель эффектов |
| `1` – `0` | Переключить эффекты 1–10 |
| `Q` `W` `E` `R` `T` | Переключить эффекты 11–15 |
| `Y` | Звук Aura Farm |
| `U` | Звук Смех |
| `I` | Звук Psi Radiation |
| `O` | Звук Don Mafia |
| `P` | Звук Грусть |
| `J` | Звук «Да уж точно» |
| `C` | Открыть / закрыть чат |
| `V` | Открыть / закрыть комментарии |
| `M` | Поставить таймкод |

> Все бинды можно изменить в настройках приложения.

## Сборка из исходников

Требования: Windows 10/11, .NET 10 SDK, WebView2 Runtime

```powershell
# Debug
dotnet build JokerDBDTracker.sln -c Debug

# Release (self-contained)
dotnet publish JokerDBDTracker.csproj -c Release -r win-x64
```
