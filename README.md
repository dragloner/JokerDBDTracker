# JokerDBDTracker

Короче, это трекер стримов JokerDBD. Нажал — смотришь, крутилки крутишь, эффекты врубаешь, кайфуешь. Без цирка, без лишней ебанины, без танцев с бубном.

Текущая версия: `v1.2.2`

## Че умеет эта шайтан-машина

- Тянет стримы с канала и показывает все по-человечески.
- Есть избранное, история, профиль, XP, уровни, престиж.
- Встроенный плеер с эффектами и биндами.
- Квесты ежедневные и недельные.
- Нормальные настройки: язык, масштаб, анимации, бинды, логирование.
- Обновление из кнопки внутри программы, без ручной ебли.

## Установка

1. Открываешь релизы: `https://github.com/dragloner/JokerDBDTracker/releases`
2. Берешь свежий `win-x64` архив.
3. Распаковываешь куда удобно.
4. Запускаешь `JokerDBDTracker.exe` и погнал.

## Обновление

- Жмешь кнопку обновления в приложении.
- Прога сама скачает, покажет прогресс/скорость, закроется и стартанет обратно.
- Если что-то пошло в жопу при применении, она поднимет текущую версию, чтоб ты не остался у разбитого корыта.

## Сборка (если ты из этих, кто любит руками)

Требуется:

- Windows 10/11
- .NET SDK
- WebView2 Runtime

Команды:

```powershell
dotnet build JokerDBDTracker.sln -c Debug
dotnet build JokerDBDTracker.sln -c Release
dotnet publish JokerDBDTracker.csproj -c Release -r win-x64
```
