# JokerDBDTracker v1.2.3

Крупное обновление удобства плеера и интерфейса: пресеты, улучшения эффектов, новые достижения/задания и серия исправлений UI/локализации.

## Кратко для релиза

- Добавлена полноценная страница пресетов в плеере (`Эффекты / Пресеты`)
- Добавлены готовые и кастомные пресеты (до 20 слотов)
- Исправлен ввод названия пресета (горячие клавиши больше не мешают печатать)
- Добавлен мини-пад для настройки центра `Fisheye`
- Улучшено управление плеером с клавиатуры при открытой панели эффектов
- Добавлен счётчик часов в профиле
- Добавлены новые достижения и задания (включая задания за использование пресетов)
- Обновлены стили интерфейса (главная / профиль / настройки / плеер)
- Исправлены проблемы `ComboBox`, локализации и ряд визуальных багов
- Исправлены баги плеера (в том числе YouTube-панель при старте и `pitch` при смене скорости)

## Для пользователей

Версия `1.2.3` делает работу с эффектами и пресетами заметно удобнее, улучшает стабильность плеера и обновляет визуальную часть приложения без потери основного функционала.
