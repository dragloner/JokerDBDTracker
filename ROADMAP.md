# JokerDBDTracker Roadmap (до 2.0)

Обновлено: 2026-02-21  
Текущая версия: `v1.2.1.1`  
Курс простой: это удобный плеер с прогрессией. Без лишней онлайновой хуйни, кроме `Watch Together`.

## `v1.3.0` — Таймкоды и заметки (вся суть плеера)

- [ ] Таймкоды: поставить, переименовать, удалить, прыгнуть в 1 клик.
- [ ] Быстрые метки (`имба`, `кринж`, `важно`, `угар`) + свои теги.
- [ ] Поиск по таймкодам/тегам/заметкам.
- [ ] Локальная подборка «лучшие моменты» по каждому стриму.
- [ ] Экспорт/импорт таймкодов в `.txt/.json`.

## `v1.4.0` — Прогрессия и статистика без мусора

- [ ] Глубокая стата просмотров: часы, сессии, серии, любимые стримы.
- [ ] Heatmap активности по дням/неделям.
- [ ] Внятная история начислений XP (за что и сколько въебало).
- [ ] Новые ачивки с реальными условиями, а не галочка ради галочки.
- [ ] Сезонный прогресс с наградами (локально, без кланового цирка).

## `v1.5.0` — Watch Together (единственный нужный онлайн)

- [ ] Комната просмотра по коду.
- [ ] Синхрон `play/pause/seek` у всех участников.
- [ ] Хост-контроль (кто рулит воспроизведением).
- [ ] Анти-рассинхрон с авто-подтяжкой времени.
- [ ] Легкий текстовый чат/реакции в комнате (минимально, без соцсети).

## `v1.6.0` — Пресеты и автоматизация плеера

- [ ] Пресеты эффектов в 1 клик.
- [ ] Профили биндов (`стрим`, `катка`, `монтаж`, свой).
- [ ] Сценарии действий: «нажал одну кнопку — плеер сам настроился».
- [ ] Умные дефолты по производительности для слабых ПК.
- [ ] Быстрый сброс к стабильному пресету, если накрутил дичь.

## `v1.7.0` — Качество просмотра и комфорт

- [ ] Гибкое управление интерфейсом плеера (compact/normal/wide).
- [ ] Полноценные темы (dark/light/custom), без поломок текста.
- [ ] Точная настройка анимаций и переходов под вкус пользователя.
- [ ] Улучшенный режим fullscreen и восстановление состояния окна.
- [ ] Нормальный центр ошибок: что сломалось и как откатиться.

## `v1.8.0` — Локальная надежность и данные

- [ ] Бэкап/восстановление профиля в 1 кнопку.
- [ ] История версий данных профиля (локальный откат).
- [ ] Оффлайн-режим каталога с кэшем стримов.
- [ ] Проверка целостности локальных данных при старте.
- [ ] Миграции профиля без “ой, всё пропало нахуй”.

## `v1.9.0` — Контент-движок внутри плеера

- [ ] Авто-сборка хайлайтов по таймкодам и активности.
- [ ] Плейлисты лучших моментов по категориям.
- [ ] Редактор нарезки (MVP) прямо в приложении.
- [ ] Шаринг подборки по ссылке/файлу.
- [ ] Импорт чужой подборки в один клик.

## `v2.0.0` — Финальная цель: охуенно удобный плеер с прогрессией

- [ ] Единое ядро: просмотр, таймкоды, прогресс, пресеты, хайлайты — всё цельно и быстро.
- [ ] Максимально предсказуемый UX: меньше действий, больше автоматизации.
- [ ] Плагинная модель для локальных расширений (без превращения в соцсеть).
- [ ] Полный рефактор тяжелых мест рендера и скролла.
- [ ] Жесткий контроль стабильности апдейтера и восстановления после сбоев.
- [ ] Полировка производительности до состояния «просто открыл и поехал».

## Сквозные принципы до 2.0

- [ ] Это плеер, а не соцпомойка.
- [ ] Онлайна минимум: только `Watch Together`.
- [ ] Любая фича должна делать просмотр удобнее, иначе нахуй.
- [ ] Никаких регрессий: если чиним одно, не ломаем три вещи рядом.
- [ ] Никакой ручной ебли в апдейтах, бэкапах и восстановлении.

Итог: меньше шума, больше пользы. Прога должна работать так, будто всё само собой,  
а ты просто смотришь стримы и качаешь прогресс без лишней головной боли.
