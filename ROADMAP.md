# JokerDBDTracker Roadmap

Обновлено: 2026-02-21  
Текущая версия: `v1.2.1.1`

## `v1.2.1.1` (выпущено, заебись)

- [x] Замедлили анимацию сворачивания, теперь видно глазами, а не как вспышка.
- [x] Апдейтер допилен: нормальный прогресс, скорость, сколько скачано и сколько всего.
- [x] Больше защиты от кривых сценариев при обновлении (локи, отвал копирования и прочая хуйня).
- [x] Код раскидан по логичным partial-файлам, мешанины меньше, читать не больно.

## `v1.2.2` (в работе, без пиздежа)

- [ ] Еще прижать микролаги на главной вкладке.
- [ ] Добить плавность анимаций во всех спорных местах.
- [ ] Подчистить хвосты по рендеру текста и hover-сценариям.
- [ ] Допроверить апдейтер на редких сетевых приколах.

## `v1.3.0` (большой кусок)

- [ ] Прокачка профиля глубже: больше наград, больше внятного прогресса.
- [ ] Доп. удобства в навигации по стримам и истории.
- [ ] Еще крепче стабильность плеера под нагрузкой.

## Техприоритеты (всегда)

- [ ] Никаких бесконечных скачиваний и зацикленных ретраев.
- [ ] Никаких "ой, файл занят" без нормального фолбэка.
- [ ] Никаких silent-падений без логов.
- [ ] Никаких регрессий, когда одно чинишь, а три вещи отваливаются к хуям.

Итог простой: меньше лагов, меньше ебли, больше нормальной работы.
