# JokerDBDTracker Roadmap (до 2.0)

Обновлено: 2026-02-21  
Текущая версия: `v1.2.1.1`  
Промежуточную мелкую полировку тут не размазываем — только жирные релизы с новыми фичами.

## `v1.3.0` — Таймкоды, заметки и нормальная движуха по стримам

- [ ] Таймкоды на стримах: ставить, редактировать, удалять, прыгать в 1 клик.
- [ ] Метки `имба`, `кринж`, `угар`, `важно` (своё тоже можно вписать).
- [ ] Поиск по таймкодам и заметкам, чтобы не рыться как в помойке.
- [ ] Локальная коллекция “лучшие моменты” по каждому стриму.
- [ ] Быстрый экспорт таймкодов в `.txt/.json`.

## `v1.4.0` — Профиль, статка и мясной прогресс

- [ ] Подробная стата просмотров: часы, дни, серии, любимые стримы.
- [ ] Нормальный heatmap активности (по дням и неделям).
- [ ] Новые ачивки не “на отъебись”, а с реальными челленджами.
- [ ] Сезонный прогресс (сброс сезона, сезонные награды).
- [ ] Вкладка “история прогресса” — когда, за что, сколько XP влетело.

## `v1.5.0` — Watch Together без ебли с настройками

- [ ] Комнаты просмотра: создал, кинул код, пацаны залетели.
- [ ] Синхрон `play/pause/seek` у всех в комнате.
- [ ] Хост-контроль (кто рулит, кто просто смотрит).
- [ ] Мини-чат и быстрые реакции в комнате.
- [ ] Анти-рассинхрон с авто-подтяжкой времени.

## `v1.6.0` — Кланы, движ и недельные зарубы

- [ ] Кланы/пати внутри приложения.
- [ ] Клановые квесты и недельные цели.
- [ ] Таблица лидеров по активности/XP/ивентам.
- [ ] Клановые роли (лидер, офицер, участник).
- [ ] Клановые “рейды” на стримы с совместными наградами.

## `v1.7.0` — Контент-движок и нарезки

- [ ] Авто-сборка “подозрительно лютых моментов” по таймкодам/меткам.
- [ ] Плейлисты лучших кусков по категориям.
- [ ] Ручной редактор нарезки внутри приложения (MVP).
- [ ] Публикация подборок в ссылку (share link).
- [ ] Импорт чужих подборок по ссылке.

## `v1.8.0` — Персонализация на максималках

- [ ] Полноценные темы (dark/light/custom) без ломания интерфейса.
- [ ] Кастомные пресеты эффектов “под один клик”.
- [ ] Профили биндов (например “стрим”, “катка”, “монтаж”).
- [ ] Гибкая плотность UI (compact/normal/wide).
- [ ] Визуальные пакеты интерфейса и иконок.

## `v1.9.0` — Облако и мультидевайс

- [ ] Облачная синхра профиля, прогресса, таймкодов и биндов.
- [ ] Логин по коду устройства (без цирка с паролями).
- [ ] Бэкап/восстановление в 1 кнопку.
- [ ] История версий профиля (откат, если всё пошло в жопу).
- [ ] Оффлайн-режим с последующей синхрой.

## `v2.0.0` — Полноценная платформа JokerDBD

- [ ] Единый центр: стримы, таймкоды, комнаты, кланы, прогресс — всё в одной экосистеме.
- [ ] Нормальный модуль рекомендаций (что смотреть дальше по твоему стилю).
- [ ] Сценарии автоматизации: “нажал один пресет — всё подстроилось”.
- [ ] Плагинная система (чтобы комьюнити могло докидывать свои приколы).
- [ ] Публичный каталог пользовательских паков/пресетов/настроек.
- [ ] Полный рефактор ядра под масштаб, чтобы не сыпалось при росте аудитории.

## Сквозные принципы до 2.0 (без розовых соплей)

- [ ] Всё должно работать из коробки: нажал — поехало.
- [ ] Никакой ручной ебли там, где можно автоматизировать.
- [ ] Любая новая фича должна быть полезной, а не “ради галочки”.
- [ ] Если что-то ломается — логи, фолбэк, восстановление, а не “ну бывает”.
- [ ] Никаких регрессий в стиле “починили кнопку, отвалился плеер к хуям”.

Итог: курс до `2.0` простой — больше реального мяса, меньше пустой болтовни,  
чтобы приложение было не игрушкой, а охуенно полезным инструментом каждый день.
