# JokerDBDTracker v1.4.0

## Кратко для релиза

- Подготовлен релиз `v1.4.0`.
- Систематизированы релизные заметки по текущему состоянию версии.
- Добавлен список известных проблем, требующих исправления в следующем патче.

## Полные патчноуты

### Основные изменения к текущей ветке релизов

- Актуализирована версия приложения до `v1.4.0`.
- Обновлена документация по релизу и состоянию функционала.

### Ранее выпущенные изменения (история)

#### `v1.3.0` — Watch Together

- Добавлена функция **Watch Together** для совместного просмотра через LAN / Radmin VPN.
- Реализованы роли хоста/гостей, синхронизация `play / pause / seek` и позиции воспроизведения.
- Добавлена автокоррекция рассинхрона позиции при расхождении более 3 секунд.
- Добавлено автооткрытие видео у гостей и применение эффектов хоста у подключённых участников.
- Добавлена локализация интерфейса и UI состояния комнаты/подключений.

#### `v1.2.3.1`

- Звуковые кнопки и горячие клавиши переведены в режим `toggle-stop`.
- Отполирован UI мини-пада `Fisheye`.
- Переработаны встроенные пресеты плеера.
- Вкладка пресетов больше не переключается автоматически на `Effects` после применения/загрузки.
- Улучшен стиль слайдеров и исправлены визуальные артефакты.
- Добавлена дополнительная логика коррекции стартовой раскладки YouTube-плеера.

#### `v1.2.3`

- Добавлена полноценная страница пресетов в плеере.
- Добавлены готовые и кастомные пресеты (до 20 слотов).
- Исправлен ввод названия пресета и конфликт с горячими клавишами.
- Добавлен мини-пад центра `Fisheye`.
- Улучшено клавиатурное управление плеером.
- Добавлены счётчик часов, новые достижения и задания.
- Обновлены стили интерфейса и исправлены баги `ComboBox` / локализации / плеера.

## Известные баги (Known Issues)

- **Некорректно сохраняются таймкоды**: после сохранения не обновляется список таймкодов.
- **Комментарии внизу плеера**: блок комментариев отображается/работает некорректно в нижней части плеера.
- **Видео не ставится на паузу по простому клику**: клик по видео не всегда переключает в паузу, как ожидается.

## Примечание

Эти известные проблемы зафиксированы в `v1.4.0` и запланированы к исправлению в следующем обновлении.
