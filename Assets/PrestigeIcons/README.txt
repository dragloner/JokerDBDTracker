Put your prestige icon files here.

Naming:
- prestige_0.png   for prestige 0-9
- prestige_1.png   for prestige 10-19
- prestige_2.png   for prestige 20-29
- ...
- prestige_10.png  for prestige 100

Supported extensions copied by project: any file type, but .png is recommended.
The app renders the icon inside a circular frame and overlays the prestige number.
